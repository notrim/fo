# 佛历应用测试指南

本指南将帮助您测试佛历应用的功能，即使您没有App开发经验。

## 步骤1：安装依赖

首先，您需要安装项目所需的依赖包。

1. 打开命令行工具（Windows上的PowerShell或cmd）
2. 导航到项目目录：
   ```
   cd d:\trae\fo
   ```
3. 安装依赖：
   ```
   npm install
   ```

## 步骤2：启动开发服务器

安装依赖后，启动Expo开发服务器：

```
npx expo start
```

执行此命令后，会出现一个二维码和一些选项。

## 步骤3：在iOS设备上运行

### 方法A：使用iOS模拟器（需要Mac电脑）

1. 在Expo开发服务器界面，按 `i` 键启动iOS模拟器
2. 模拟器会自动打开并运行佛历应用

### 方法B：使用真实iOS设备

1. 在iPhone或iPad上安装 **Expo Go** 应用（从App Store下载）
2. 打开Expo Go应用，使用相机扫描命令行中显示的二维码
3. 应用会自动加载并运行

## 步骤4：测试功能

### 测试核心功能：

1. **查看日历**：
   - 滑动日历查看不同月份
   - 观察佛教节日是否被标记（有特殊颜色）

2. **查看节日详情**：
   - 点击标记的节日日期
   - 确认底部是否显示节日详情

3. **浏览节日列表**：
   - 向下滚动查看完整的佛教节日列表
   - 确认所有节日信息是否正确显示

4. **验证节日数据**：
   - 检查释迦牟尼诞辰（佛诞节）是否正确标记
   - 验证其他主要佛教节日的日期和描述

## 常见问题解决

### 依赖安装失败
- 确保网络连接正常
- 尝试使用管理员权限运行命令行
- 清除npm缓存：`npm cache clean --force`

### 无法启动开发服务器
- 检查端口是否被占用
- 尝试重启电脑后再试

### 应用无法在iOS设备上运行
- 确保设备与电脑在同一WiFi网络
- 检查Expo Go应用是否为最新版本
- 尝试重新扫描二维码
